# 2dCard

A node that can be used to create 2D cards for card games

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/2dCard.git
```

## Documentation
https://github.com/ChillCube/2dCard/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon 2dCard
```

## For Developers
```bash
push-all-addons
```
